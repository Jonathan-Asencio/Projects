Battleship

PURPOSE OF PROJECT: Simulate the game of Battleship using TCP/IP

VERSION or DATE: 20190712

HOW TO START THIS PROJECT: Run the main() method for 'GameClient' once connected the game will begin.

AUTHORS: Jonathan Asencio

USER INSTRUCTIONS: Run the main() method for 'GameClient'. Using a device that is on the same network, run the 'GameServer' once the two devices are connected the game will begin.

How To Play:
Similar to the board game of Battleship each player will take turns guessing the location of the other players battleships. Each player's board and battleship location is automatically generated.
